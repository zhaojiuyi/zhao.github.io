scrapy简介

中文文档

下图显示了Scrapy架构及其组件的概述，以及系统内部发生的数据流的概要（以红色箭头显示）。

提供了这些组件的简要说明以及有关它们的详细信息的链接。数据流也描述如下。

  • Scrapy Engine(引擎): 负责Spider、ItemPipeline、Downloader、Scheduler中间的通讯，信号、数据传递等。

  • Scheduler(调度器): 它负责接受引擎发送过来的Request请求，并按照一定的方式进行整理排列，入队，当引擎需要时，交还给引擎。

  • Downloader（下载器）：负责下载Scrapy Engine(引擎)发送的所有Requests请求，并将其获取到的Responses交还给Scrapy Engine(引擎)，由引擎交给Spider来处理，

  • Spider（爬虫）：它负责处理所有Responses,从中分析提取数据，获取Item字段需要的数据，并将需要跟进的URL提交给引擎，再次进入Scheduler(调度器).

  • **Item、Pipeline (管道)**：它负责处理Spider中获取到的Item，并进行进行后期处理（详细分析、过滤、存储等）的地方。

  • Downloader Middlewares（下载中间件）：你可以当作是一个可以自定义扩展下载功能的组件。

  • Spider Middlewares（Spider中间件）：你可以理解为是一个可以自定扩展和操作引擎和Spider中间通信的功能组件（比如进入Spider的Responses;和从Spider出去的Requests）

准备工作

爬取内容： 智联招聘信息

地址： https://www.zhaopin.com/ https://sou.zhaopin.com/

爬取过程：先通过第一个网址拿到工作岗位和城市，然后在第二个网址进行招聘信息的搜索，爬取，随后存储到mysql数据库。

网站分析：

打开第一个网站，选择一个招聘岗位，点击进入。

注意看网址，域名后面有一个jl=601和kw=Java开发，前面的601是城市编号，后面的就是工作岗位了，我们把网址上的601改成具体的城市名称，发现也可以拿到信息，那就节省了很多时间。回到首页，可以看到列出了很多工作岗位和城市（首页往下翻就能看到城市）。

所以，只需要在首页拿到工作岗位和城市名称，然后拼接成一个搜索链接，我们就能拿到想要的招聘信息。

爬取岗位和城市

在项目目录下打开终端，输入scrapy genspider getjobsinfo 'www.zhaopin.com'

可以看到在spiders包下创建了一个getjobsinfo的pyhthon文件，这就是刚刚创建的爬虫。

编写代码如下

from scrapy import Request, Selector

class GetjobsinfoSpider(scrapy.Spider):
    name = 'getjobsinfo'
    allowed_domains = ['zhaopin.com']

    # start_urls = ['https://www.zhaopin.com/']
    def start_requests(self):
        #  提交爬取路径交给引擎，开始爬取
        yield Request(url='https://www.zhaopin.com/')

    def parse(self, response, **kwargs):
        #  拿到响应体 使用xpath解析数据
        jobs_list = Selector(text=response.text).xpath(
            "//a[@class='zp-jobNavigater__pop--href']/text()").extract()  # 工作列表
        citys_list = Selector(text=response.text).xpath(  #   城市列表
            "//div[@class='footerFuncCity clearfix']/ul/li/strong/a/text()").extract()  
        print(jobs_list)
        print(citys_list)

中间件集成selenium

第一个页面不是动态的，所以可以直接爬取，但是第二个页面是动态的，所以需要使用selenium帮助我们爬取。

可以选择在中间件进行集成。打开middlewares.py，编辑SipderworkscrapyDownloaderMiddleware类。修改如下内容。

实现思路：拿到响应体后，使用BeautifulSoup解析出网页的文本，如果文本数量小于200，就使用selenium重新爬取。

先要在虚拟环境中安装BeautifulSoup和selenium，同时将Chrome驱动放入虚拟环境下的python根目录。

pip install BeautifulSoup -i https://pypi.tuna.tsinghua.edu.cn/simple

pip install selenium -i https://pypi.tuna.tsinghua.edu.cn/simple

from scrapy.http import HtmlResponse
from selenium import webdriver
from selenium.common import TimeoutException
from bs4 import BeautifulSoup

class SipderworkscrapyDownloaderMiddleware:

    def __init__(self):
        # 在初始化方法中创建Chrome实例
        options = webdriver.ChromeOptions()
        options.add_argument('--headless')  # 设置无界面
        self.webdriver = webdriver.Chrome(options=options)

    def __del__(self):
        self.webdriver.close()  # 关闭窗口
        self.webdriver.quit()  # 关闭浏览器

    def process_response(self, request, response, spider):
        try:
            # 提取响应体文本
            pure_text = BeautifulSoup(response.body).get_text()
            if len(pure_text) < 200:
                print('Chrome driver begin...')
                self.webdriver.get(url=response.url)
                # wait = WebDriverWait(self.webdriver, timeout=20)
                return HtmlResponse(url=response.url, body=self.webdriver.page_source,
                                    encoding='utf-8')  # 返回selenium渲染之后的HTML数据
            else:
                return response
        except TimeoutException:
            return HtmlResponse(url=response.url, encoding='utf-8', status=500)
        finally:
            print('Chrome driver end...')

中间件修改完成后在settings.py中设置使用我们修改过的中间件， 设置里默认有写，取消注释即可，SipderworkscrapyDownloaderMiddleware是中间件的类名。

DOWNLOADER_MIDDLEWARES = {
   'sipderWorkScrapy.middlewares.SipderworkscrapyDownloaderMiddleware': 543,
}

爬取招聘信息

遍历上面得到的岗位列表和城市列表，拼接url，提交给引擎进行下一步爬取。

for job in jobs_list:
    for city in citys_list:
        # 提交一个Request给引擎
        yield Request(url=f'http://sou.zhaopin.com/?jl={city}&kw={job}')

这里有一个问题，下载到的页面还会使用当前的parse方法解析数据，这并不是我们所期望的，所以要在这里添加一个回调，使用其他方法解析这个Request，所以需要再写一个回调方法，使用该回调方法解析下一步的数据。

同时在该回调方法里解析数据，然后用item接收。

修改getjobsinfo.py的代码为

import scrapy
from scrapy import Request, Selector

from sipderWorkScrapy.items import JobInfo


class GetjobsinfoSpider(scrapy.Spider):
    name = 'getjobsinfo'
    allowed_domains = ['zhaopin.com']

    # start_urls = ['https://www.zhaopin.com/']
    def start_requests(self):
        yield Request(url='https://www.zhaopin.com/')

    def parse(self, response, **kwargs):
        jobs_list = Selector(text=response.text).xpath(
            "//a[@class='zp-jobNavigater__pop--href']/text()").extract()  # 工作列表
        citys_list = Selector(text=response.text).xpath(
            "//div[@class='footerFuncCity clearfix']/ul/li/strong/a/text()").extract()  # 工作地点
        print(jobs_list)
        print(citys_list)

        for job in jobs_list:
            for city in citys_list:
                url = f'http://sou.zhaopin.com/?jl={city}&kw={job}'
                yield Request(url=url, callback=self.jobs_parse)

    def jobs_parse(self, response):
        doms = Selector(text=response.text).xpath(
            "//*[@id='positionList-hook']/div/div[@class='joblist-box__item clearfix']").extract()
        for dom in doms:
            job_name = Selector(text=dom).xpath(
                "//span[@class='iteminfo__line1__jobname__name']/@title").extract_first()  # 岗位名称
            job_salary = Selector(text=dom).xpath(
                "//p[@class='iteminfo__line2__jobdesc__salary']/text()").extract_first()  # 薪资范围
            job_place = Selector(text=dom).xpath(
                "//ul[@class='iteminfo__line2__jobdesc__demand']/li[1]/text()").extract_first()  # 工作地点
            job_experience = Selector(text=dom).xpath(
                "//ul[@class='iteminfo__line2__jobdesc__demand']/li[2]/text()").extract_first()  # 经验要求
            job_education = Selector(text=dom).xpath(
                "//ul[@class='iteminfo__line2__jobdesc__demand']/li[3]/text()").extract_first()  # 学历要求
            job_tag = Selector(text=dom).xpath(
                "//div[@class='iteminfo__line3__welfare']/div/text()").extract()  # 岗位标签
            company_name = Selector(text=dom).xpath(
                "//span[@class='iteminfo__line1__compname__name']/@title").extract_first()  # 公司名称
            company_type = Selector(text=dom).xpath(
                "//div[@class='iteminfo__line2__compdesc']/span[1]/text()").extract_first()  # 企业类型
            company_scale = Selector(text=dom).xpath(
                "//div[@class='iteminfo__line2__compdesc']/span[2]/text()").extract_first()  # 企业规模
            link = Selector(text=dom).xpath(
                "//a[@class='joblist-box__iteminfo iteminfo']/@href").extract_first()  # 岗位链接
            job_info = JobInfo()
            job_info['job_name'] = job_name
            job_info['job_salary'] = job_salary
            job_info['job_place'] = job_place
            job_info['job_experience'] = job_experience
            job_info['job_education'] = job_education
            job_info['job_tag'] = job_tag
            job_info['company_name'] = company_name
            job_info['company_type'] = company_type
            job_info['company_scale'] = company_scale
            job_info['link'] = link
            # 将数据提交
            yield job_info

保存招聘信息

使用peewee保存招聘信息，在管道进行处理。

创建一个Model.py编写代码如下

from peewee import *

db = MySQLDatabase('数据库名称',
                   host="主机地址",
                   port=3306,  # 端口 默认3306
                   user="账户名",
                   password="密码")


class DataModel(Model):
    class Meta:
        database = db


class JobsInfo(DataModel):  # 招聘岗位
    job_name = CharField(max_length="255", verbose_name="岗位名称")
    job_salary = CharField(max_length="255", verbose_name="薪资范围")
    job_place = CharField(max_length="255", verbose_name="工作地点")
    job_experience = CharField(max_length="255", verbose_name="经验要求")
    job_education = CharField(max_length="255", verbose_name="学历要求")
    job_tag = TextField(default="", verbose_name="岗位标签")
    company_name = CharField(max_length="255", verbose_name="公司名称")
    company_type = CharField(max_length="255", verbose_name="公司类型")
    company_scale = CharField(max_length="255", default="", verbose_name="企业规模")
    link = TextField(default="", verbose_name="岗位链接")


db.create_tables([JobsInfo])

打开piplines.py，编辑如下信息

from sipderWorkScrapy.Model import JobsInfo


# 使用mysql存储招聘数据
class JobsInfoPipline:
    def __init__(self):
        pass

    def close_spider(self, spider):
        pass
	
    # 处理提交的item
    def process_item(self, item, spider):
        try:
            job_info = JobsInfo()
            job_info.job_name = item['job_name']
            job_info.job_salary = item['job_salary']
            job_info.job_place = item['job_place']
            job_info.job_experience = item['job_experience']
            job_info.job_education = item['job_education']
            job_info.job_tag = item['job_tag']
            job_info.company_name = item['company_name']
            job_info.company_type = item['company_type']
            job_info.company_scale = item['company_scale']
            job_info.link = item['link']
            job_info.save()
            print(f"{item['job_name']}保存成功")
        except (IndexError, TypeError, TimeoutError):
            print("保存失败")

最后在settings.py下启动这个管道。

ITEM_PIPELINES = {
   'sipderWorkScrapy.pipelines.JobsInfoPipline': 300,
}
